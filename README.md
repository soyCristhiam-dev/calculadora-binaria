# Calculadora Binaria (Django)


Proyecto de ejemplo en Django que convierte números binarios a decimal, octal y hexadecimal.


## Requisitos
- Python 3.10+ (o 3.8+)
- pip
- virtualenv (recomendado)


## Instalación local


```bash
# clonar el repositorio
git clone <TU-REPO-URL>.git
cd calculadora_binaria_django


# crear y activar virtualenv
python -m venv venv
# Windows
venv\Scripts\activate
# Linux / Mac
source venv/bin/activate


# instalar dependencias
pip install -r requirements.txt


# ejecutar migraciones y levantar servidor
python manage.py migrate
python manage.py runserver
